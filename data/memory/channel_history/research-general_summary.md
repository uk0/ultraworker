# research-general 채널 분기별 요약

## 채널 개요
Sionic AI 리서치팀의 주요 연구 토론 채널. LLM, RAG, 임베딩, 멀티모달 모델, 한국어 NLP 등 최신 AI 연구 동향을 공유하고 논의하는 공간.

---

## 2023-Q4 (10월~12월) — 메시지 약 88건

### 주요 토픽
- **MoE (Mixture of Experts) 모델**: Mixtral 8x7B 출시 및 MoE 아키텍처 분석, 전문가 라우팅 메커니즘 논의
- **OpenAI DevDay**: GPT-4 Turbo, Assistants API, 128K 컨텍스트 윈도우 등 발표 내용 분석
- **RAG (Retrieval-Augmented Generation)**: RAG 파이프라인 개선, 청킹 전략, 벡터 DB 비교
- **Whisper v3**: 음성 인식 모델 성능 평가, 한국어 인식 품질 테스트
- **Google Gemini**: 멀티모달 성능 벤치마크, GPT-4V 대비 비교
- **LLM 병합 기법**: 모델 병합(TIES, DARE 등) 실험 및 결과 공유
- **환각(Hallucination) 탐지**: 사실 검증 기법, RAG 기반 환각 감소 방법

### 주요 참여자
- **Sigrid Jin (진형 박)** (52건): 가장 활발한 기여자, 논문/기술 동향 공유 주도
- **박우명** (12건): 기술적 논의 참여
- **Noah (고석현)** (12건): 연구 방향 논의
- **song** (9건): 논문 리뷰 참여

### 주요 결정/이벤트
- RAG 파이프라인 아키텍처 초기 설계 방향 논의
- MoE 모델의 실무 활용 가능성 검토

## 2024-Q3 (7월~9월) — 메시지 약 223건

### 주요 토픽
- **Reranker 평가**: 다양한 리랭커 모델(Cross-encoder, ColBERT 등) 벤치마크 비교
- **ColBERT/XTR 연구**: 후기 상호작용(late interaction) 기반 검색 모델 심층 분석
- **Llama 3.1 405B**: 대규모 오픈소스 모델의 실무 배포 가능성 검토
- **한국어 OCR 모델**: 문서 이미지 인식 모델 개발 및 평가
- **하이브리드 검색(Hybrid Search)**: BM25 + 벡터 검색 결합 전략 최적화
- **프롬프팅 서베이**: 체계적인 프롬프트 엔지니어링 방법론 정리
- **알리바바 Qwen팀 미팅**: 외부 연구팀과의 기술 교류
- **Meta Llama 경진대회**: 국내 LLM 관련 대회 참가
- **HCLT 논문 제출**: 한국어 컴퓨터 언어학 학회 논문 작성
- **SAE (Sparse Autoencoder) 실험**: LLM 내부 표현 분석
