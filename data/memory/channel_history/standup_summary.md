# standup (C058VN1LUKE) - 스탠드업 채널 요약

> 채널 기간: ~2025-Q2 ~ 현재 (활성, API 최대 10,000건 기준)
> 총 메시지: 10,000건+ (API 한도로 인해 2025-Q2 이후만 수집됨)
> 의미 있는 메시지: 약 5,060건
> 주요 참여자: 전 직원 (30~41명, 지속 증가)
> 채널 목적: 일일 출퇴근/일정/위치 공유, 재택/외근/휴가 알림

---

## 채널 운영 방식

### 자동화 봇
1. **이모지 붙여조잉 봇**: 매일 자동으로 출근 확인 메시지 발송 (`<!channel>`)
   - 직원들이 이모지를 붙여 출근 확인
2. **휴가/반차 알림 봇**: 당일 휴가자 자동 공지
   - 예: `2026-01-02 (Fri) - :palm_tree: [김덕현] 휴가`

### 메시지 패턴
- 재택근무 알림 (가장 빈번, 분기당 300~400건)
- 외근/미팅 알림 (분기당 60~110건)
- 휴가/반차 알림 (분기당 70~90건)
- 건강 관련 (병원, 컨디션) (분기당 40~60건)
- 개인 일정 (점심 약속, 커피챗 등)
