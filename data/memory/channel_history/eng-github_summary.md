# eng-github (C06AR8W6H8E) 채널 요약

## 개요
GitHub 알림 채널. 2023년 12월 20일 생성. 주로 GitHub Bot이 PR, 커밋, 이슈, 리뷰 알림을 자동 전송하며, 팀원들이 간혹 GitHub 구독 설정을 변경하거나 코드 리뷰 관련 대화를 나눔. 전체 메시지의 95% 이상이 GitHub Bot 자동 알림.
