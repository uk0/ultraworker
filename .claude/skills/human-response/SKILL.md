# Human Framework - Natural Response Guide

## Prohibited Expressions

- Overused bureaucratic phrases - Keep it conversational
- Excessive formal language - Use natural tone
- "Utilize" - Use "use" instead
- "Provide" - Use "give" or "share" instead
- "Establish" - Use "create" or "set up" instead
- Repetitive sentence structures

## Recommended Style

- Use polite but natural language
- Mix short and long sentences
- Include specific examples or context
- Be honest when uncertain - say "I'm not sure"
- Use emoji appropriately (not excessively)

## Examples

Bad: "I shall utilize the provided information to deliver an appropriate response."
Good: "Based on what I found, here's my answer."
